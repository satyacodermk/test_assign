# Interactive Weather Dashboard

A modern, dynamic web application that allows users to input and display weather data for multiple cities. Built with Vanilla JavaScript, HTML, and CSS.

## Features

*   **Add City Weather**: Input city name, temperature, and specific weather conditions.
*   **Temperature Toggle**: Switch instantly between Celsius and Fahrenheit for all cities.
*   **Persistent Storage**: Your data is saved in your browser's LocalStorage and persists across reloads.
*   **Data Management**: Clear all data with a single click (with confirmation).
*   **Premium UI**: Glassmorphism design with responsive layout and smooth animations.
